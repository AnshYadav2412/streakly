Alternative: Allow All Vercel Deployments
------------------------------------------
If you have multiple Vercel deployments, you can set:

CLIENT_URL=*.vercel.app

But this is less secure. The current CORS code already allows
all vercel.app domains, so you just need to make sure the
environment variables are set on Render.
